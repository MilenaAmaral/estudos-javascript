# 🚀 Meus Estudos de JavaScript

Este repositório foi criado para documentar a minha evolução, prática e aprendizado na linguagem JavaScript. Aqui reúno exercícios de lógica de programação, desafios práticos e conceitos fundamentais da linguagem.

## 📁 Estrutura do Repositório

O projeto está organizado em arquivos específicos para cada conceito estudado:

* `tipos.js`: Estudos sobre tipos de dados (Strings, Numbers, Booleans, etc.).
* `operadoresMatematicos.js` e `operadoresComparacao.js`: Prática com operações aritméticas e relacionais.
* `combinandoComparadores.js`: Exercícios práticos combinando múltiplos comparadores lógicos.
* `hoisting.js`: Entendimento do comportamento de elevação de variáveis e funções no JS.
* `legibilidade.js`: Práticas para escrita de código limpo e de fácil compreensão.
* `desafio.js`: **Calculadora de Trocos** (Desafio prático que simula a lógica de compra, valor pago e cálculo de troco formatado com `.toFixed(2)`).

## 🛠️ Tecnologias Utilizadas

* **JavaScript (ES6+)**
* **Node.js** (para execução dos scripts no terminal)
* **Visual Studio Code** (IDE de desenvolvimento)

## 🎯 Objetivo

Consolidar uma base sólida em lógica de programação e versionamento de código, preparando o terreno para o desenvolvimento de aplicações web completas (Full-Stack).
